# 字幕组机翻小助手 （使用谷歌翻译）
免费开源工具，不带广告。    
翻译视频时，字幕每个汉字都手打太累了，机翻虽然不正确但工作量变成了删字，比起打字轻松不少。

## 如何使用
拖入 .srt 或 .ass 字幕文件，翻译字幕文件。
