### 解决什么问题?
用机器翻译整个字幕文件。用户只需要把文件拖进去即可。
加快翻译字幕的速度  

### 适合谁用?
字幕组

### 如何使用?
1. 拖入字幕文件( .srt .ass 后缀的文件)
2. 点击"开始翻译"        
3. 1~2秒后就会翻译完成，自动在文件夹中显示          

### 重要说明
**自备科学上网工具**，因为用了 Google 翻译。    
如果点"开始翻译"没反应就是网络出错了

3、谷歌可能改接口了导致程序失效
