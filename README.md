## 字幕组机翻小助手 :fire:
字幕翻译时，每个汉字都手打太累了，机翻虽然不正确但工作量变成了删字，轻松不少。          
 （使用谷歌翻译）（免费开源，不带广告）        

<img src="./image/1.png" alt="how the app look like" width="380">


### 使用方法
拖入或选择 .srt .ass 后缀的字幕文件，点击绿色按钮"开始翻译"        
1~2秒后就会翻译完成，自动在文件夹中显示。
